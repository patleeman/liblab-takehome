# Developing

To develop his library:

1. Copy `.env.sample` to `.env` then add your API key.
2. Create a virtual env `python -m venv .venv`
3. Activate venv `source .venv/bin/activate`
4. Install libraries `pip install -r requirements.txt`

Run tests:
```bash
./test.sh
```
